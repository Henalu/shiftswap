# Roadmap — ShiftSwap MVP (30 días)

## Semana 1: Setup y Autenticación
- Día 1-2: Inicializar Next.js, Tailwind, shadcn/ui, Supabase
- Día 3-4: Schema de base de datos + migraciones
- Día 5-7: Auth completo (registro, login, logout, middleware protección rutas)

## Semana 2: Turnos (CRUD)
- Día 8-9: Formulario de publicación de turno
- Día 10-11: Lista de turnos disponibles con filtros
- Día 12-13: Detalle de turno + botón "Me interesa"
- Día 14: Sistema de estados del turno

## Semana 3: Chat y Matching
- Día 15-16: Sistema de solicitudes (crear, aceptar, rechazar)
- Día 17-18: Notificaciones in-app
- Día 19-21: Chat en tiempo real (Supabase Realtime)

## Semana 4: Confirmación y Polish
- Día 22-23: Flujo de confirmación de intercambio
- Día 24-25: Generación de documento PDF
- Día 26-27: UI polish, responsive, edge cases
- Día 28-29: Testing manual + fixes
- Día 30: Deploy a producción (Vercel + Supabase Cloud)

## Post-MVP
- Panel de supervisor
- App móvil / PWA
- Integraciones con RRHH
- Analíticas
- Multi-idioma
